Load Data from Webpage

I downloaded the fire.csv file into the same folders as my Notebook file, loaded the data into the firedf object in the Notebook (with read.csv()), and showed the stucture of the file with str() (because showing the whole file was too big). From the structure one can see that there are 48 individuals (R calls them observations which it abbreviates with “obs.”), 6 variables, and the variables are named sttype, tslf, tdw, litdep, fuel1h, and tltd.
