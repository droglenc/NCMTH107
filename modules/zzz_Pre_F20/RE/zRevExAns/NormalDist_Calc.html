Situation 1

  1. The percentage of X<0.11 is 54.4.
  2. The percentage of X>-0.11 is 54.4.
  3. The percentage of -1.45<X<1.11 is 79.3.
  4. The X that has 33% of Xs smaller is -0.44.
  5. The X that has 25$ of Xs larger is 0.67.
  6. The most common 90% of Xs are between -1.64 and is 1.64.

R Appendix

library(NCStats)
( q1 <- distrib(0.11,mean=0,sd=1) )
( q2 <- distrib(-0.11,mean=0,sd=1,lower.tail=FALSE) )
( q3ab <- distrib(1.11,mean=0,sd=1) )
( q3a <- distrib(-1.45,mean=0,sd=1) )
